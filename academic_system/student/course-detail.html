Course Overview

This course provides an in-depth study of advanced algorithms and data structures. Students will learn to analyze algorithmic complexity, implement efficient solutions, and understand the mathematical foundations of computer science.

Learning Objectives

  • Analyze time and space complexity of algorithms
  • Implement dynamic programming solutions
  • Design graph algorithms for real-world problems
  • Apply advanced data structures effectively

Prerequisites

  • Data Structures (CS 201)
  • Discrete Mathematics (MATH 150)
  • Programming Fundamentals (CS 100)

Course Topics

Week 1-4: Algorithmic Analysis

  • • Big O, Omega, and Theta notation
  • • Master theorem applications
  • • Recurrence relations

Week 5-8: Dynamic Programming

  • • Optimal substructure
  • • Memoization techniques
  • • Classic DP problems

Week 9-12: Graph Algorithms

  • • Shortest path algorithms
  • • Network flow problems
  • • Minimum spanning trees

Week 13-16: Advanced Topics

  • • String algorithms
  • • Computational geometry
  • • NP-completeness
